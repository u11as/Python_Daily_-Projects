## About

This repository contains beginner-level Python scripts that cover essential programming concepts such as data types, mathematical operations, type conversion, and basic project development.

The goal of this repository is to practice Python consistently and strengthen problem-solving skills step by step.

## Projects Included

- Understanding different Python data types
- Using the type() function
- Working with strings, integers, floats, and booleans

### Mathematical Operations
- Arithmetic operators
- Order of operations (PEMDAS)
- Expression evaluation

### Number Manipulation
- Working with numerical calculations
- Using the round() function
- Performing arithmetic expressions

### Type Checking and Conversion
- Taking user input
- Using len() function
- Type conversion (int(), float(), str())
- Understanding type errors

### Tip Calculator Project
A simple real-world mini project that:
- Takes user input
- Calculates tip percentage
- Splits the total bill among people
- Rounds the final amount
- Displays formatted output using f-strings
